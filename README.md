# 🎓 Cerevo – AI Career Guidance and Recommendation System

🚀 **Cerevo** is an AI-powered career recommendation platform designed to help students explore suitable career paths based on their interests, preferences, and academic background.

The system integrates **Artificial Intelligence, recommendation techniques, and structured career roadmaps** to provide personalized career guidance for students.

# 📊 Project Overview

Many students face difficulties when selecting career paths due to lack of awareness about available opportunities.

Cerevo addresses this challenge by providing an **AI-driven career guidance platform** that helps students:

- Explore career domains
- Receive personalized career recommendations
- Understand required skills and educational pathways
- Generate structured career roadmaps
- Download career guidance reports

# 🚀 How to Run the Project

## Step 1 – Clone Repository


git clone https://github.com/GudaluruChandana/Cerevo-Career-Guidance-and-Recommendation-System.git


---

## Step 2 – Run Backend


cd backend
pip install -r requirements.txt
uvicorn main:app --reload


Backend runs at:


http://localhost:8000


---

## Step 3 – Run Frontend


cd frontend
npm install
npm run dev


Frontend runs at:


http://localhost:5173
